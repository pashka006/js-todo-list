# Simple To-Do App

Компактный веб-додаток для управління завданнями, написаний на чистому JavaScript.

## 🛠 Технології
* HTML5 / CSS3
* Vanilla JavaScript (ES6+)
* LocalStorage API

## ✨ Особливості
* Додавання та видалення завдань.
* Захист від введення порожніх рядків.
* Збереження списку справ у пам'яті браузера (LocalStorage) — дані не зникають після оновлення сторінки.
